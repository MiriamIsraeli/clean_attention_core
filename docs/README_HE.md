# Clean Attention - ליבת מערכת ביטול הרעשים

##  תיאור כללי
מודול זה מכיל את **אלגוריתמי הליבה** של מערכת Clean Attention לביטול רעשים
סטציונריים (מזגן, מנוע, זמזום חשמלי) בזמן אמת.

המערכת מזהה רעשים מחזוריים קבועים, מייצרת **גל הפוך** (Anti-Noise) ומבטלת אותם
באמצעות עקרון ההתאבכות ההורסת.

## האלגוריתמים - הסבר מפורט

### 1. מסנן LMS (Least Mean Squares)
**הקבצים:** `lms_filter.h`, `lms_filter.cpp`, `algorithms.py`

**מה עושה:** לומד את דפוס הרעש בזמן אמת ומייצר אות ביטול.

**איך עובד:**
- מקבל **אות ייחוס** (הרעש מהמיקרופון) ו**אות רצוי** (האות המעורב)
- מחשב פלט: `y = w^T * x` (מכפלה של משקולות ודגימות)
- מחשב שגיאה: `e = desired - y`
- מעדכן משקולות: `w = w + μ * e * x`
- ככל שעובר זמן, המסנן מתכנס ו-`y` מתקרב לרעש האמיתי
- הגל ההפוך הוא: `anti_noise = -y`

**פרמטרים חשובים:**
- `filterLength` (N): מספר המשקולות. ערך גבוה = דיוק רב, חישוב כבד
- `stepSize` (μ): קצב למידה. גדול = מהיר אך לא יציב. קטן = איטי אך מדויק

---

### 2. מסנן FxLMS (Filtered-x LMS)
**הקבצים:** `fxlms_filter.h`, `fxlms_filter.cpp`, `algorithms.py`

**מה עושה:** גרסה מתקדמת של LMS המותאמת ל-ANC (ביטול רעשים אקטיבי).

**מה שונה מ-LMS:**
- מתחשב ב**נתיב המשני** - המסלול האקוסטי מהרמקול לאוזן
- הקלט מסונן דרך מודל הנתיב המשני לפני עדכון המשקולות
- מבטיח שהגל הנגדי יגיע **בפאזה הנכונה** לביטול מדויק

**שימוש:**
- חיוני כשיש רכיב חומרה (רמקול + מיקרופון)
- ה-`secondaryPath` מייצג את התגובה האקוסטית של החלל

---

### 3. גלאי רעש סטציונרי (Noise Detector)
**הקבצים:** `noise_detector.h`, `noise_detector.cpp`, `algorithms.py`

**מה עושה:** מזהה אילו תדרים באות הם רעש קבוע (סטציונרי).

**איך עובד:**
1. מחלק את האות ל**חלונות זמן** (frames)
2. מפעיל **חלון Hanning** למניעת דליפה ספקטרלית
3. מחשב **FFT** לכל חלון - מקבל ספקטרום תדרים
4. עוקב אחרי **ממוצע ושונות** של כל תדר לאורך זמן
5. תדרים עם **מקדם שונות נמוך** (יציבים) = רעש סטציונרי
6. מחזיר **מסכה**: `true` = רעש, `false` = אות רצוי

**סף היציבות (threshold):**
- ערך נמוך (0.1) = רגישות גבוהה, יזהה יותר כרעש
- ערך גבוה (0.5) = שמרני, רק רעש מאוד יציב

---

### 4. FFT (Fast Fourier Transform)
**ממומש בתוך:** `noise_detector.cpp`

אלגוריתם Cooley-Tukey Radix-2 להתמרת פורייה מהירה.
מאפשר ניתוח ספקטרלי יעיל של אותות שמע (מרחב הזמן → מרחב התדר).

## פירוט הקוד - C++

### lms_filter.cpp - עיבוד דגימה

```cpp
double LMSFilter::process(double reference, double desired) {
    // 1. הכנסת דגימה חדשה לבאפר
    for (int i = N_ - 1; i > 0; --i)
        x_[i] = x_[i - 1];
    x_[0] = reference;

    // 2. מכפלה פנימית: y = w · x
    double y = inner_product(w_.begin(), w_.end(), x_.begin(), 0.0);

    // 3. חישוב שגיאה
    lastError_ = desired - y;

    // 4. עדכון משקולות (כלל LMS)
    for (int i = 0; i < N_; ++i)
        w_[i] += mu_ * lastError_ * x_[i];

    return y;
}
```

| שורות | תפקיד |
|-------|--------|
| 2-4 | הזזת באפר - כל דגימה זזה מקום אחד ימינה, החדשה נכנסת בהתחלה |
| 7 | חישוב הפלט - מכפלה פנימית של וקטור המשקולות בוקטור הדגימות |
| 10 | השגיאה - ההפרש בין מה שאנחנו רוצים למה שקיבלנו |
| 13-14 | עדכון משקולות - כל משקולת מתעדכנת פרופורציונלית לשגיאה ולדגימה |

### noise_detector.cpp - ניתוח מסגרת

```cpp
void NoiseDetector::analyzeFrame(const vector<double>& frame) {
    // חלון Hanning + FFT
    for (int i = 0; i < fftSize_; ++i) {
        double window = 0.5 * (1.0 - cos(2*PI*i / (fftSize_-1)));
        data[i] = complex(frame[i] * window, 0.0);
    }
    fft(data);

    // עדכון סטטיסטיקות לכל תדר
    for (int i = 0; i < halfN; ++i) {
        sumMag_[i] += abs(data[i]);
        sumMagSq_[i] += abs(data[i])^2;
    }
}
```

| 7 | ביצוע FFT - המרה ממרחב הזמן למרחב התדר |

## פירוט הקוד - Python

### algorithms.py - מחלקת LMSFilter

```python
def process(self, reference, desired):
    for i in range(n):
        self.x = np.roll(self.x, 1)  # הזזת באפר
        self.x[0] = reference[i]     # דגימה חדשה
        y = np.dot(self.w, self.x)   # מכפלה פנימית
        e = desired[i] - y           # שגיאה
        self.w += self.mu * e * self.x  # עדכון
```

```python
def simulate():
```
