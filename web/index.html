ℹ️ איך עובדת המערכת

עיקרון ANC: שני ערוצים מדמים שני מיקרופונים באוזנייה:

1Primary (מיקרופון פנימי): רעש + דיבור = d[n]
2Reference (מיקרופון חיצוני): רעש בלבד + דיליי קטן = x[n]
3NLMS לומד: y[n] = wᵀ·x[n] (אמידת רעש)
4VAD מזהה דיבור → מקפיא עדכון w כדי לא לבטל דיבור
5e[n] = d[n] - y[n] → מתכנס לדיבור בלבד
6Anti-Noise = -y[n] → הגל שיושמע מהאוזניות (ביטול פיזי)

⏱️ שלב 1 — יצירת קובץ Reference (דיליי)

העלה את הקובץ המקורי. המערכת תיצור עותק עם דיליי קטן המדמה את הפרש ההגעה בין המיקרופונים (~1.5 ס"מ = 2 דגימות ב-44100Hz).
